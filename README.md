# Pathfinding Algorithms Visualizer

An interactive simulator for visualizing different pathfinding algorithms. You can place walls, set start and end points, and watch the algorithms work through the grid.

**Live demo:**  
https://dallss.github.io/pathfinding-algorithms-visualizer/

This project also helped me improve my understanding of vanilla HTML, CSS, and JavaScript. I used JavaScript to create custom elements and manage all interactions and animations without any frameworks.

## Features

- Supports multiple algorithm instances running side by side
- Interactive grid: click and drag to draw walls
- Adjustable animation speed
- Reset and clear functions
- Algorithms implemented:
  - Dijkstra's Algorithm
  - A* Search
  - Breadth-First Search
  - Depth-First Search

## How to Run Locally

Clone the repo and open `index.html` in your browser. No build tools or installations required.

```bash
git clone https://github.com/dallss/pathfinding-algorithms-visualizer.git
